# SkillSphere

AI-powered Hyperlocal Freelance Ecosystem built using MERN Stack.

## Features
- Multi-role Authentication
- Freelancer Dashboard
- Client Dashboard
- Admin Panel
- AI Job Matching
- Proposal & Bidding System
- Contracts & Payments
- Real-time Chat
- Notifications
- Analytics Dashboard

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Redux Toolkit
- Socket.IO
- Tailwind CSS
