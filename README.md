# 🚀 SkillSphere – Intelligent Hyperlocal Freelance Ecosystem

SkillSphere is a full-stack MERN-based freelance marketplace that connects clients and freelancers through an AI-powered hyperlocal ecosystem.

🌐 **Live Demo:** https://skill-sphere-tanisha10.vercel.app  
💻 **GitHub Repository:** https://github.com/tanisha2601/SkillSphere

---

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/tanisha2601/SkillSphere.git
cd SkillSphere
```

---

## Backend Setup

```bash
cd server
npm install
npm run dev
```

Create `.env`

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
CLIENT_URL=http://localhost:5173
```

---

## Frontend Setup

```bash
cd client
npm install
npm run dev
```

Create `.env`

```env
VITE_APIBASEURL=http://localhost:5000/api
VITE_SOCKET_URL=http://localhost:5000
```
